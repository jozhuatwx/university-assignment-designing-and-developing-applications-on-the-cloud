# Public Bus Management System
An online portal to manage buses, stations, routes, users, and payments using Azure.

## Technologies
Frontend
* Angular
* TypeScript
* SCSS
* Azure Static Web App

Backend
* Azure App Service
* ASP.NET Core 5, C#
* Entity Framework Core
* Azure SQL Database
* Azure Storage (Blob & Table)
